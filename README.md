# ✅ To-Do List App

A modern **To-Do List Web Application** built using **React** and **Tailwind CSS** to help users manage daily tasks efficiently with a clean and responsive interface.

🌐 **Live Demo:**
https://to0-do0-list.netlify.app/

---

## 🚀 Features

* ✅ Add new tasks
* ✏️ Edit existing tasks
* 🗑️ Delete tasks
* ✔️ Mark tasks as completed
* 📱 Responsive design
* ⚡ Fast performance using React
* 🎨 Modern UI powered by Tailwind CSS
* 🔔 Snackbar notifications for actions

---

## 🛠️ Tech Stack

* **React**
* **Vite**
* **Tailwind CSS**
* **JavaScript (ES6+)**
* **Context API**
* **Local Storage**

---

## 📦 Installation

Clone the repository:

```bash
git clone https://github.com/YOUR_USERNAME/to-do-list.git
```

Go to project folder:

```bash
cd to-do-list
```

Install dependencies:

```bash
npm install
```

Run development server:

```bash
npm run dev
```

---

## 🏗️ Build for Production

```bash
npm run build
```

Preview production build:

```bash
npm run preview
```
